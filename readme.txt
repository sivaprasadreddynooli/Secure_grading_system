The project is developed using Python version 3.5.2 and flask web framwework

To start the project run the file 'app.py' and the website will run in http://127.0.0.1:5000/  by default.
the login page will prompt for the username and password and thus take you to tour to the portal based on the student
or the faculty you selected

All mailings will happen from studentacademyportal@gmail.com for the links of forgot password

The database used is SQLite3 with the mozilla firefox extension
